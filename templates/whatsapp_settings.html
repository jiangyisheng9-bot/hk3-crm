🌐 让 webhook 可被 Meta 访问（ngrok）

CRM 跑在 localhost:5001，Meta 无法直接访问。用 ngrok 暴露一个公网 HTTPS URL：

# 安装 ngrok（一次性）
brew install ngrok/ngrok/ngrok

# 启动隧道（保持运行）
ngrok http 5001

# 复制 https://xxxx.ngrok-free.app/whatsapp/webhook
# 填到 Meta Webhook 配置页

或者直接跑 bash start_with_ngrok.sh 一键启动 CRM + ngrok。

📋 在 Meta 后台需要做的

  1. 登录 developers.facebook.com → 创建/选择 App → 添加 WhatsApp 产品
  2. API Setup 页面：复制 Phone Number ID 和 WABA ID
  3. Business Settings → System Users → 创建 System User → 生成永不过期 token（权限 whatsapp_business_messaging、whatsapp_business_management）
  4. WhatsApp → Configuration → Webhook：粘贴上面的 URL + 自定义 Verify Token，点击 Verify
  5. Subscribe webhook fields：勾选 messages
  6. 回到本页 → 填表 → 测试连接 → 启用
